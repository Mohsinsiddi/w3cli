# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-18

### Added
- Initial release of w3cli
- Wallet management (create, import, list, remove, switch active)
- Balance checking across 26+ EVM chains and testnets
- Smart contract interaction (read/write functions, ABI management)
- ERC-20 token deployment
- Transaction history and details lookup
- ENS name resolution
- Gas price estimation
- RPC endpoint management with benchmarking and smart selection
- Network switching (mainnet/testnet) for all supported chains
- Cross-platform support (macOS, Linux, Windows)
- OS keychain integration for secure key storage
- Interactive TUI for contract interactions
- Checksum address validation
- ABI encoding/decoding utilities
- Keccak256 hashing
- Function selector computation
- Unit conversion (wei/gwei/ether)
